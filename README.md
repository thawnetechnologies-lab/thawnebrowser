THAWNE BROWSER - Navegador Web Integrado
========================================

Descripción:
------------
Thawne Browser es un navegador web completamente integrado desarrollado en Python
con PyQt5 y QtWebEngine. No requiere navegadores externos, todo funciona dentro
de la aplicación.

Características:
----------------
✓ Motor Chromium integrado
✓ Navegación por pestañas múltiples
✓ Historial completo
✓ Barra de direcciones inteligente
✓ Interfaz moderna y profesional
✓ Completamente offline

Instalación:
------------
1. Ejecutar el instalador ThawneBrowser_Setup.exe
2. Seguir las instrucciones del asistente de instalación
3. El navegador se instalará en Program Files

Requisitos del sistema:
-----------------------
- Windows 7 o superior (64-bit recomendado)
- 4 GB de RAM mínimo
- 500 MB de espacio en disco
- Conexión a Internet para navegar

Desinstalación:
---------------
- Ir a Panel de Control > Programas y características
- Seleccionar "Thawne Browser" y hacer clic en Desinstalar
- O usar el enlace en el menú Inicio

Soporte:

Para problemas o sugerencias, contactar al desarrollador.
